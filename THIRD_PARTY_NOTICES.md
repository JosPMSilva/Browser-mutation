## Runtime Dependencies

Browser Mutation does not bundle npm packages or third-party runtime libraries.

The current source install requires the user's own `node` executable to run the local MCP server and collector scripts.

## Target Application Assets

Browser Mutation can inspect icon names, SVG attributes, class names, and styles from a user-owned local development page. Those captured details remain part of the user's target application and are not redistributed by Browser Mutation.
